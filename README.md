# Academic Resume Website

A comprehensive, professional academic resume website designed specifically for researchers, professors, and academic professionals. Built with HTML, CSS, and JavaScript, featuring content-focused design and academic-specific sections.

## 🎓 Academic Features

### Content-Focused Sections
- **Research Profile** - Detailed research interests and academic background
- **Education** - Complete academic history with dissertation/thesis details
- **Research Experience** - Academic positions and research roles
- **Publications** - Organized by type (journal articles, book chapters, conferences)
- **Teaching Experience** - Philosophy, courses taught, and student supervision
- **Grants & Awards** - Research funding and academic honors
- **Professional Service** - Editorial, organizational, and university service

### Academic-Specific Elements
- **Citation Counts** - Display impact metrics for publications
- **ORCID Integration** - Link to academic identifier
- **Google Scholar Profile** - Direct link to citation profile
- **Detailed Timelines** - Comprehensive academic history
- **Grant Information** - Funding amounts, agencies, and roles
- **Student Supervision** - Current and completed advisees

## 🚀 Features

- **Responsive Design** - Optimized for all devices and screen sizes
- **Print-Friendly** - Professional printing with proper page breaks
- **Sidebar Navigation** - Easy access to all sections
- **Minimal Animations** - Content-focused with subtle interactions
- **Academic Icons** - Font Awesome icons tailored for academia
- **Professional Typography** - Clean, readable fonts suitable for academic content

## 🛠️ Customization Guide

### 1. Personal Information
Update the following in `index.html`:

```html
<!-- Sidebar Header -->
<h1 class="name">Your Full Name</h1>
<h2 class="title">Assistant Professor | PhD in [Field]</h2>

<!-- Contact Information -->
<span>University Name, Department</span>
<span>your.email@university.edu</span>
```

### 2. Research Profile
Replace placeholder content in the About section:
- Research focus areas
- Publication count
- Funding amounts
- Research interests list

### 3. Publications
Format your publications using academic citation style:

```html
<li>
    <strong>Your Name</strong>, Co-Author, A., & Co-Author, B. (2024). 
    "Paper Title." <em>Journal Name</em>, 45(2), 123-145. 
    DOI: 10.1000/journal.2024.123456
    <span class="citation-count">[Cited by 15]</span>
</li>
```

### 4. Academic Links
Update important academic profiles:
- ORCID ID: `https://orcid.org/0000-0000-0000-0000`
- Google Scholar: `https://scholar.google.com/citations?user=XXXXX`
- Personal website: `https://yourwebsite.com`

### 5. Color Scheme
The academic theme uses professional colors:
- Primary: `#3498db` (blue) - for main elements
- Secondary: `#27ae60` (green) - for teaching sections
- Accent: `#f39c12` (orange) - for grants and awards
- Service: `#9b59b6` (purple) - for service sections

## 📚 Content Guidelines

### Publications Section
- **Order**: Reverse chronological (newest first)
- **Format**: Use standard academic citation format for your field
- **Categories**: Separate journal articles, book chapters, and conference presentations
- **Citation Counts**: Include if available and relevant to your field

### Teaching Philosophy
- Keep concise (2-3 sentences)
- Focus on pedagogical approach
- Mention specific methods or technologies used
- Highlight inclusive teaching practices

### Grant Information
- Include funding amounts (if appropriate in your field)
- Specify your role (PI, Co-PI, etc.)
- List collaborators when relevant
- Include both current and completed grants

### Service Activities
- **Editorial Service**: Journal roles and review work
- **Professional Organizations**: Committee work and leadership
- **University Service**: Internal committee assignments

## 🖨️ Print Optimization

The resume is optimized for printing:
- **Page Breaks**: Sections avoid breaking across pages
- **Font Sizes**: Readable when printed
- **Color Handling**: Important elements print well in grayscale
- **Layout**: Sidebar information remains accessible

To print:
1. Use the "Download Resume" button in the sidebar
2. Or press `Ctrl+P` (Windows) / `Cmd+P` (Mac)
3. Select "More settings" → "Print backgrounds" for best results

## 🌐 Deployment Options

### GitHub Pages (Free)
1. Create a GitHub repository
2. Upload your files
3. Go to Settings → Pages
4. Select source branch
5. Access at `https://yourusername.github.io/repository-name`

### Academic Institution Hosting
Many universities provide web hosting for faculty:
1. Check with your IT department
2. Upload files to your university web space
3. Access via `https://university.edu/~yourusername`

### Custom Domain
1. Purchase a domain (e.g., `yourname.com`)
2. Use services like Netlify, Vercel, or GitHub Pages
3. Point your domain to the hosting service

## 📝 Academic Best Practices

### Content Tips
1. **Be Specific**: Include concrete details (publication counts, funding amounts)
2. **Stay Current**: Update regularly with new achievements
3. **Quantify Impact**: Use metrics where appropriate
4. **Professional Tone**: Maintain formal academic language
5. **Proofread**: Ensure accuracy in all citations and dates

### SEO for Academics
- Use your full name in the title tag
- Include your field/discipline in meta descriptions
- Use structured data for publications
- Link to your institutional profile
